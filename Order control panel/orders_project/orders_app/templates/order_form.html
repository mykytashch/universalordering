Инструкция по отправке заказов на нашу панель через API

Для того чтобы отправить заказ на нашу панель, выполните следующие шаги:

  1. Подготовьте данные заказа в формате JSON.
  2. Отправьте HTTP POST-запрос на адрес http://127.0.0.1:8000/create_order/ с данными заказа в теле запроса.
  3. Обработайте ответ сервера.

Пример на Python:

import requests

data = {
  "id": null,
  "customer_name": "Ваше имя",
  "product": "Название продукта",
  "created_at": "Дата создания",
  "updated_at": "Дата обновления"
}

response = requests.post('http://127.0.0.1:8000/create_order/', json=data)

if response.status_code == 200:
    print("Заказ успешно отправлен!")
else:
    print("Ошибка при отправке заказа:", response.text)
